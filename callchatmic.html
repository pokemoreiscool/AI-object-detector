Can't run from a local file

Browsers block microphone access and WebSocket connections when you open an HTML file directly (file://).

You need to serve it over HTTP. The easiest way:

Option 1 — Python (built-in, no install needed)

cd path/to/your/folder python -m http.server 8080 → open http://localhost:8080/voicelink.html

Option 2 — Node: npx serve . then open the URL it shows.
Option 3 — Upload to any web host (Netlify, GitHub Pages, etc.)
