⚠️ Must be served over HTTP

You opened this as a file:// URL. Browsers block microphone and WebSocket access from local files.

Run this in the same folder:

python -m http.server 8080

Then open:

http://localhost:8080/voicelink.html

Or: npx serve .  •  Upload to Netlify / GitHub Pages
