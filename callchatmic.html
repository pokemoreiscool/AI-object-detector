⚠️ Opening as a local file won't work Browsers block microphone access on file:// URLs. Serve it locally instead:

1. Open a terminal in the same folder
2. Run: python -m http.server 8080
3. Open: http://localhost:8080/voicelink.html

Or upload the file to any web host (GitHub Pages, Netlify, etc.)
